# KYDEN KK | Data Analyst Portfolio

A modern, high-impact data analyst portfolio designed for **KYDEN KK**. This single-page website showcases data projects with a focus on clean aesthetics and user interaction.

## 🚀 Features
- **Dual-Theme System**: Seamlessly toggle between a "Modern Turquoise" (Dark) theme and a "Light Pink" (Light) theme.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.
- **Project Showcase**: A 3-column responsive grid featuring key data analysis projects with tool tags.
- **Custom Typography**: Features *Lobster Two* for headings and *Inter* for body text via Google Fonts.

## 🛠️ Technical Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with CSS Variables for dynamic theming.
- **JavaScript**: Minimal vanilla JS for theme switching logic.

## 🐢 Theme Details
- **Turquoise (Default)**: Deep charcoal background with turquoise accents. Icon: 🐢.
- **Pink**: Lavender blush background with light pink accents. Icon: 🐻.

## 📂 Project Structure
- `index.html`: Main site structure.
- `style.css`: Comprehensive styling and theme definitions.
- `script.js`: Theme toggle logic.
- `plan/`: Original project design and planning documents.

## 🌐 Deployment
This portfolio is ready for hosting on **GitHub Pages**. 
1. Push the code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` folder.
4. Your site will be live at `https://<username>.github.io/<repo-name>/`.
